## Application and dev stack overview

It is a Conway's game of life implementation with JavaScript in browser. TypeScript is used for static type checking and esbuild for bundling.

## Running instructions

```bash
# Build
npm install
npm run build

# Open in browser
chromium index.html

# With webserver
npx serve -s .
```
